# 🤖 Echo AI Chatbot (Android)

Echo AI is an **AI-powered chatbot Android application** built using **Java in Android Studio**. The application provides an interactive chat interface where users can ask questions and receive intelligent responses powered by modern AI models or APIs.

The project demonstrates how AI chat systems can be integrated into **mobile applications** to create smart assistants and conversational interfaces.

---

# 📱 Features

* 🤖 AI-powered chatbot interaction
* 💬 Real-time messaging interface
* 📲 Mobile-friendly chat UI
* ⚡ Fast response generation
* 🔗 API integration with AI services
* 🧠 Natural language conversation support
* 📋 Chat history display

# ⚙️ Installation

### 1. Clone the Repository

```bash
git clone https://github.com/yourusername/EchoAI-Chatbot.git
```

---

### 2. Open in Android Studio

1. Launch **Android Studio**
2. Click **Open Project**
3. Select the cloned repository folder

---

### 3. Build and Run

1. Connect an **Android device** or start an **Android emulator**
2. Click **Run ▶**
3. The Echo AI Chatbot app will launch.

---

# 🔑 API Configuration

To enable AI responses, add your API key inside the app code.

Example:

```java
OPENAI_API_KEY = "your_api_key_here";
```

Make sure the API key is used securely when making requests to the AI service.
